================================================================================
                      正確的 MyGene 查詢方法
================================================================================

【推薦方法】
  import mygene
  mg = mygene.MyGeneInfo()
  result = mg.querymany(['SQSTM1'],
                        scopes='symbol',
                        species='human',
                        fields='ensembl.gene,symbol,entrezgene')
  
  預期結果: ENSG00000161011

【常見錯誤】
  ✗ 缺少 species='human' 參數
  ✗ 未指定 scopes='symbol'
  ✗ 使用別名而非官方符號
  ✓ 始終驗證 Ensembl ID

================================================================================
                      為什麼 MyGene 找不到它?
================================================================================

【問題 1】未指定物種
  解決: 加上 species='human' 參數

【問題 2】查詢範圍不正確
  解決: 使用 scopes='symbol'

【問題 3】使用不被識別的別名
  解決: 使用官方符號 SQSTM1

【問題 4】版本不同步
  解決: pip install --upgrade mygene

查詢失敗時,詳見 SQSTM1_FAQ.md 的問題 1-3
