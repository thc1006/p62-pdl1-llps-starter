# Cell-based validation SOP (outline)
- Lines: A549/H1975 (NSCLC); controls per design.
- Constructs: p62 ΔUBA/ΔPB1/S403 mutants; STUB1 ΔU-box/ΔTPR; PD-L1 Δtail and K→R mutants.
- Assays: Co-IP; ubiquitination (K48/K63); internalization/recycling (Rab5/7/11); lysosomal routing (HIP1R OE/KO).
- LLPS imaging: live confocal + FRAP; concentration/temperature dependence; reversible tests beyond hexanediol.
- Stats: power, replicates, blinding; preregistration of contrasts.
